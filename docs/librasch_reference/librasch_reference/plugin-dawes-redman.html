dawes-redman

Name

dawes-redman -- calculate FHR variations using the Dawes/Redman criteria

Description

The plugin calculates the fetal heart rate variations using the Dawes/Redman criteria. The estimation of the baseline of the fetal heart rate is done according the descriptions in the various papers from Dawes and Redman. For the digital filter, a Butterworth filter with order 4 and corner-frequency of 0.0017Hz (approx. 0.1 min-1)is used (using the 3.75sec epoch-values results in a samplerate of approx. 0.2667Hz). The filter was designed on the website of Tony Fisher, using the mkfilter cgi-script. This can be found at http://www-users.cs.york.ac.uk/~fisher/mkfilter/ .
