ra_ev_set_value

Name

ra_ev_set_value --  set the property of an event

Synopsis

LIBRAAPI int ra_ev_set_value (prop_handle proph, long event, double value);

Arguments

proph

event-property handle

event

number of the event

value

property-value

Description

This function sets the event-property proph of the event event to value.

event.c
