ra_eval_loadxml

Name

ra_eval_loadxml --  load evalution in XML format

Synopsis

LIBRAAPI int ra_eval_loadxml (meas_handle mh, const char * file);

Arguments

mh

measurement handle

file

name of the evaluation file (optional)

Description

This function loads the evaluation(s) that belongs to the measurement mh. If no filename file is given, the default-filename will be used.

source file

load_xml.c
