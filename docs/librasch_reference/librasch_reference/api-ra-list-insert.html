ra_list_insert

Name

ra_list_insert --  insert an entry to a linked list

Synopsis

int ra_list_insert (void ** *head, void * prev, void * item);

Arguments

*head

prev

previous item

item

item to be added

Description

This function inserts 'item' in a linked list (identified by 'head'). The item will come directly after 'prev'.

source file

linked_list.c
