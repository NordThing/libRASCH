read-rri

Name

read-rri -- Handle RR interval-files (nearly the final design)

Description

The plugin provides access to RR interval files. It now supports the first RR-interval format specific for libRASCH. The design of the format is not fixed and will be definitely changed in the not so distant future.

Version

0.1.0

Author(s)

Raphael Schneider (rasch@med1.med.tum.de)
