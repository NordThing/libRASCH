respiration

Name

respiration -- calculates respiration parameters

Description

The plugin calculates parameters from a respiration signal. At the moment only chest measurement based respiration signals can be processed. At the moment only the mean of the chest measurement of the last inter beat interval is calculated. In the future other measures (e.g. freq. of respiration) will be implemented.
